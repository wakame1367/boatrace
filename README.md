# BoatRace

[Boatrace official HP - download](https://www.boatrace.jp/owpc/pc/extra/data/download.html)

### Prerequisites

At first [pipenv](https://github.com/pypa/pipenv) install!!

```
pip install pipenv
```

## Getting Started

```
git clone https://github.com/wakamezake/boatrace.git
pipenv install
```
